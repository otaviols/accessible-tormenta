> "Novos caminhos para os aventureiros mais audaciosos de Arton"

## Visão Geral

**Heróis de Arton** é o primeiro suplemento oficial de Tormenta 20, expandindo as opções de criação de personagem e regras do livro básico. Este livro apresenta:

- **5 Novas Raças**: Duende, Eiradaan, Galokk, Meio-Elfo e Sátiro
- **Nova Classe**: Treinador, mestre de companheiros animais
- **14 Classes Variantes**: Versões alternativas das classes básicas
- **30 Novas Origens**: Históricos adicionais para personagens
- **Centenas de Novos Poderes**: Para todas as classes e categorias
- **38 Distinções**: Sistema de conquistas e títulos de prestígio
- **Arsenal Expandido**: Novos equipamentos, magias e itens mágicos
- **Regras Opcionais**: Sistemas avançados para campanhas

---

## Estrutura do Suplemento

Este suplemento está organizado em 4 capítulos principais:

### [Capítulo 1: Campeões de Arton](01-campeoes-arton/)

Novas opções de raças, classes, origens e poderes para expandir a criação de personagens.

**Conteúdo:**
- Novas Raças jogáveis
- Classe Treinador
- Classes Variantes
- Novas Origens
- Novos Poderes (de Classe e Gerais)
- Tabelas para Personagens

### [Capítulo 2: Distinções](02-distincoes/) ✅

Sistema de conquistas e títulos especiais que personagens podem alcançar através de feitos heroicos.

**Conteúdo:**
- Sistema de Distinções (Admissão, Marca, Poderes)
- 36 Distinções completas (de Aeronauta Goblin a Vigarista)
- Sistemas especiais (Ornitópteros, Pontos de Insanidade, etc.)

### [Capítulo 3: Arsenal dos Heróis](03-arsenal-herois/)

Novos equipamentos, bases, magias e itens mágicos para aventureiros.

**Conteúdo:**
- Novos Equipamentos (armas, armaduras, itens)
- Sistema de Bases (cômodos e mobílias)
- Novas Magias Arcanas
- Novos Itens Mágicos e Artefatos

### [Capítulo 4: Regras Opcionais](04-regras-opcionais/)

Sistemas avançados e variantes de regras para campanhas mais complexas.

**Conteúdo:**
- Atributos Variados, Raças Abertas, Devoções Abertas
- Complicações, Idades Variadas, Objetivos Heroicos
- Combate Avançado, Culinária Avançada
- Sistema de Domínios (gestão de reinos)
- Exploração de Masmorras

---

## Compatibilidade

Todo o conteúdo deste suplemento é **100% compatível** com o livro básico de Tormenta 20. As novas opções podem ser integradas gradualmente em campanhas existentes ou usadas desde o início em novas aventuras.
