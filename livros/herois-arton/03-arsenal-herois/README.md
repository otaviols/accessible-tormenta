## Usando Este Capítulo

Os equipamentos e itens neste capítulo expandem as opções do livro básico:

- **Equipamentos Mundanos**: Podem ser comprados normalmente
- **Bases**: Ideais para campanhas de longa duração
- **Magias**: Adicionadas às listas de conjuradores
- **Itens Mágicos**: Distribuídos como tesouro pelo mestre
- **Artefatos**: Elementos de trama para campanhas épicas
