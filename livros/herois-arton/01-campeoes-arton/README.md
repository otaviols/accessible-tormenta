# Capítulo 1: Campeões de Arton

> "Heróis surgem de todas as raças, classes e origens. Seus caminhos são tão diversos quanto suas histórias."

[← Voltar ao Heróis de Arton](../README.md) | [Índice Principal](../../../README.md)

---

## Visão Geral

Este capítulo expande dramaticamente as opções de criação de personagens em Tormenta 20, oferecendo:

- **5 Novas Raças** para jogar
- **1 Nova Classe** completa (Treinador)
- **14 Classes Variantes** como alternativas às classes básicas
- **30 Novas Origens** para histórico de personagens
- **Centenas de Novos Poderes** para todas as categorias
- **Tabelas de Personalização** para nomes, aparência, trejeitos e objetivos

## Usando Este Capítulo

As opções apresentadas neste capítulo são **totalmente compatíveis** com o livro básico. Você pode:

- Criar personagens usando apenas conteúdo deste capítulo
- Combinar raças/classes novas com origens do livro básico
- Adicionar novos poderes a personagens existentes (com aprovação do mestre)
- Usar classes variantes como alternativas às classes básicas
